CSY1018 Internet Technology

This module provides students with the essential knowledge necessary to develop and implement a well-designed client-side dynamic website. Starting with an overview of the Internet technologies, the overall software architecture of the Internet including servers, clients, browsers, the module covers HTML, Cascading Style Sheets (CSS), and JavaScript in detail. The main focus is the code underlying client-side dynamic Web software. Principles of design, including usability and accessibility, are introduced
